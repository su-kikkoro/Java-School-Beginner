ComprehensiveBeginner01_02

class ComprehensiveBeginner01_03
3人（man / woman / child）のじゃんけん結果を乱数で生成し、勝者を判定して表示するプログラム。
全員同じ手、または全員違う手の場合は「あいこ」とし、それ以外は「同じ手の2人 vs 違う手の1人」で勝敗判定を行う。
勝敗判定は isWin(a, b) メソッドで行い、a が b に勝つ場合に true を返す。
